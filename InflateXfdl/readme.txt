* InflateXfdl: uncompress IBM Forms .xfdl/.xfdd file into clear-text XML

* USAGE:
  1) Install: drag/drop InflateXfdl.jar to your Windows Desktop
  2) Execute: <double-click>InflateXfdl desktop icon, and select an IBM forms file to uncompress
  
* CHANGES:
  9/02/2013:
  - Include .xfdd, xfdl in default search list 
  - Clear display pane when new XFDL file is selected
  - Allow reading uncompressed XFDL, as well as bas64-gzip, files
